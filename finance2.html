                <h2 class="text-2xl font-bold mb-6 text-[#007BFF]">Introduction : Le Rôle des Marchés Financiers et Concepts Fondamentaux</h2>

                <p class="mb-4">Pour comprendre les marchés financiers, il est essentiel de saisir leur rôle central dans l'économie et les concepts qui les sous-tendent. Un marché financier est un lieu (physique ou virtuel) où se rencontrent l'offre et la demande de capitaux. Son rôle principal est d'allouer efficacement le capital.</p>

                

                <div class="space-y-4">

                    <div>

                        <h3 class="text-xl font-semibold mb-2">Concepts Économiques et Financiers Fondamentaux</h3>

                        <ul class="list-disc list-inside space-y-3 pl-4">

                            <li><strong>L'Offre et la Demande :</strong> Le prix des actifs financiers est déterminé par l'équilibre entre l'offre (quantité de titres disponibles) et la demande (intérêt des investisseurs). <span class="text-sm text-gray-600">Ex: Si beaucoup de gens veulent acheter une action (forte demande) et peu veulent la vendre (faible offre), son prix monte.</span></li>

                            <li><strong>La Liquidité :</strong> Facilité avec laquelle un actif peut être acheté ou vendu sans affecter significativement son prix. <span class="text-sm text-gray-600">Ex: Les actions des grandes entreprises sont généralement très liquides.</span></li>

                            <li><strong>Le Risque et le Rendement :</strong> Généralement, plus un investissement est potentiellement rentable (rendement élevé), plus il est risqué. <span class="text-sm text-gray-600">Ex: Les actions peuvent offrir de forts rendements mais sont plus risquées que les obligations d'État.</span></li>

                            <li><strong>La Volatilité :</strong> Mesure de l'ampleur des variations de prix d'un actif. Une forte volatilité signifie que le prix peut changer rapidement. <span class="text-sm text-gray-600">Ex: Les actions des jeunes entreprises technologiques sont souvent plus volatiles.</span></li>

                            <li><strong>La Diversification :</strong> Répartir ses investissements sur différents actifs pour réduire le risque global. <span class="text-sm text-gray-600">Ex: Ne pas mettre tous ses œufs dans le même panier.</span></li>

                            <li><strong>L'Inflation :</strong> Augmentation générale des prix, réduisant le pouvoir d'achat. Les investissements doivent viser un rendement supérieur à l'inflation. <span class="text-sm text-gray-600">Ex: Si l'inflation est de 2%, un placement qui rapporte 1% fait perdre du pouvoir d'achat.</span></li>

                        </ul>

                        <h3 class="text-xl font-semibold mb-2">Actions</h3>

                        <p><strong>Définition :</strong> Titre de propriété représentant une fraction du capital d'une entreprise. Devenir actionnaire = co-propriétaire.</p>

                        <p><strong>Comment ça rapporte ?</strong> Plus-value (vente plus chère que l'achat), dividendes (part des bénéfices, non garantis).</p>

                        <p><strong>Risques :</strong> Perte en capital, volatilité, faillite de l'entreprise.</p>

                        <p class="text-sm text-gray-600 mt-1"><em>Exemple : Acheter 10 actions "TechInnov" à 100€. Si le cours monte à 120€, plus-value de 20€/action.</em></p>

                    </div>

                    <div data-tab-content="obligations" class="produits-tab-content p-2">

                        <h3 class="text-xl font-semibold mb-2">Obligations</h3>

                        <p><strong>Définition :</strong> Titre de créance. Vous prêtez de l'argent à une entité (État, entreprise) contre un intérêt (coupon) et remboursement du capital à l'échéance.</p>

                        <p><strong>Comment ça rapporte ?</strong> Coupons réguliers, remboursement du capital, plus-value sur marché secondaire.</p>

                        <p><strong>Risques :</strong> Risque de taux d'intérêt, risque de crédit (défaut de l'émetteur), risque d'inflation.</p>

                        <p class="text-sm text-gray-600 mt-1"><em>Exemple : Obligation d'État 1000€, taux 3%, échéance 5 ans. Vous recevez 30€/an et 1000€ à la fin.</em></p>

                    </div>

                    <div data-tab-content="derives" class="produits-tab-content p-2">

                        <h3 class="text-xl font-semibold mb-2">Produits Dérivés (Contrats)</h3>

                        <p><strong>Définition :</strong> Instruments dont la valeur "dérive" de celle d'un actif sous-jacent (action, indice, matière première...). Souvent utilisés pour spéculation ou couverture. Très risqués.</p>

                        <ul class="list-disc list-inside pl-4 space-y-1 mt-1">

                            <li><strong>Futures (Contrats à terme) :</strong> Engagement d'acheter/vendre un actif à un prix et une date futurs.</li>

                            <li><strong>Options :</strong> Droit (non obligation) d'acheter (call) ou vendre (put) un actif à un prix d'exercice avant une échéance.</li>

                            <li><strong>CFD (Contracts For Difference) :</strong> Spéculer sur la variation de prix sans détenir l'actif. Effet de levier important.</li>

                        </ul>

                         <p class="text-sm text-red-600 font-semibold mt-1"><em>Attention : Produits complexes et à haut risque, réservés aux investisseurs avertis.</em></p>

                    </div>

                     <div data-tab-content="autresProduits" class="produits-tab-content p-2">

                        <h3 class="text-xl font-semibold mb-2">Autres Produits Courants</h3>

                         <ul class="list-disc list-inside pl-4 space-y-1 mt-1">

                            <li><strong>Fonds d'Investissement (OPCVM, ETF) :</strong> Paniers de titres gérés par des professionnels. Les ETF (Trackers) reproduisent un indice. Permettent une diversification instantanée.</li>

                            <li><strong>Matières Premières :</strong> Or, pétrole, etc. Investissement via contrats dérivés ou ETF spécialisés.</li>

                            <li><strong>Devises (Forex) :</strong> Marché d'échange des monnaies. Très liquide mais volatil.</li>

                        </ul>

                    </div>

                            <li><strong>Position Short (Vente à Découvert) :</strong> Parier sur la baisse (plus risqué).</li>

                            <li><strong>Stop Loss :</strong> Ordre pour vendre et limiter les pertes. Essentiel !</li>

                            <li><strong>Take Profit :</strong> Ordre pour vendre et prendre ses bénéfices.</li>

                            <li><strong>Effet de Levier :</strong> Investir plus que son capital. Amplifie gains ET pertes. À utiliser avec extrême prudence.</li>

        // Helper functions for indicators

        function calculateMA(data, period) {

            if (data.length < period) return [];

            const ma = [];

            for (let i = 0; i < period -1; i++) ma.push(null); // Fill initial with null

            for (let i = period - 1; i < data.length; i++) {

                let sum = 0;

                for (let j = 0; j < period; j++) {

                    sum += data[i - j];

                }

                ma.push(sum / period);

            }

            return ma;

        }


        function calculateRSI(prices, period = 14) {

            const rsi = [];

            if (prices.length <= period) return [];


            let gains = 0;

            let losses = 0;


            // Calculate initial average gain and loss

            for (let i = 1; i <= period; i++) {

                const diff = prices[i] - prices[i-1];

                if (diff > 0) gains += diff;

                else losses -= diff; // losses are positive values

            }

            let avgGain = gains / period;

            let avgLoss = losses / period;


            if (avgLoss === 0) rsi.push(100);

            else {

                const rs = avgGain / avgLoss;

                rsi.push(100 - (100 / (1 + rs)));

            }

            

            // Calculate subsequent RSI values

            for (let i = period + 1; i < prices.length; i++) {

                const diff = prices[i] - prices[i-1];

                let currentGain = 0;

                let currentLoss = 0;

                if (diff > 0) currentGain = diff;

                else currentLoss = -diff;


                avgGain = (avgGain * (period - 1) + currentGain) / period;

                avgLoss = (avgLoss * (period - 1) + currentLoss) / period;

                

                if (avgLoss === 0) rsi.push(100);

                else {

                    const rs = avgGain / avgLoss;

                    rsi.push(100 - (100 / (1 + rs)));

                }

            }

            return rsi;

        }
